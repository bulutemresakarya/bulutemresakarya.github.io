Dört Bileşen, Tek Marka

Koyu lacivert + turuncu ikilisi mobil uygulamada ve yönetim panelinde birebir aynı hex kodlarıyla kullanılıyor; bot da bu kimliği "kaptan" hitabı ve tutarlı emoji diliyle sohbete taşıyor.

BACKEND

Node.js + Express + PostgreSQL

Tek süreçte bot, mobil API ve admin API. Cron görevleri: kota, süre, temizlik, kampanya worker'ı.

WHATSAPP BOTU

Baileys + Meta Cloud API + OpenAI

Grup dinleme ayrı, DM ayrı kanal. Structured Outputs ile niyet analizi; rate-limit ve karaliste koruması.

MOBİL UYGULAMA

Expo / React Native

OTP giriş (WhatsApp kodu), rol bazlı onboarding, teklif sistemi, finans takibi, Expo push, uygulama içi abonelik.

YÖNETİM PANELİ

Vue 3 + PrimeVue

Kullanıcı/ilan/belge onayı, sistem metinleri düzenleme, bot QR & yeniden bağlanma kontrolü.

🧠 Serbest metin → yapılandırılmış ilan

"Ankara'ya 25 ton demir, tenteli lazım acil" gibi dağınık grup mesajları AI ile güzergâh, tonaj, araç tipi ve fiyata ayrıştırılır.
